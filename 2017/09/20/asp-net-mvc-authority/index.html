asp.net mvc Library project needs authority—————————

1.Create folder named Authority.

2.Create class named CheckLogin.cs

3.Create class named AuthAttribute.cs. In this class, we will override the method named OnActionExecuting().

4.In the UsersController.cs, we should create cookie for current user after confirming that whether UserName and Password are valid.

5.Logout Method

6.Before the method that we want to add authority, we should add [Auth(Code =”libraryadmin,admin”)].

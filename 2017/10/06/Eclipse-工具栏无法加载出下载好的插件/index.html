Eclipse 工具栏无法加载出下载好的插件

Veröffentlicht am 2017-10-06

问题描述

软件测试中有一个可以查看覆盖率的插件Eclemma，下载完之后在Eclipse的工具栏中发现并没有显示这个插件。

解决方案

Eclipse -> window -> Customize Perspective， 勾选上 Eclemma。

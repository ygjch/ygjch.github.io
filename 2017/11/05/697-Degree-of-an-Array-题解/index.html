697. Degree of an Array 题解

Veröffentlicht am 2017-11-05

题意

一个给定数组，求包含出现次数最多元素的最短子序列长度。

题解

要求这个最短子序列要包含出现次数最多的元素。就要找到这个出现次数最多的元素，然后在给定数组中找到第一次和最后一次出现的位置，
然后两个位置下标做差加一即为最短子序列长度。

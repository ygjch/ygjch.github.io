605. Can Place Flowers 题解

Veröffentlicht am 2017-11-09

题意

一花坛，1 代表已经栽好花， 0 代表该位置还没有栽花。

给定一数字 n， 问能否在该花坛中栽 n 株花。

栽花需要符合 相邻位置不栽花 的原则。

题解

从 0 位置开始遍历， 直到最后一位。 在当前位置满足 前后两个位置和当前位置均为0的时候，该位置可栽花 flower[i] = 1，并且栽花数 cnt++。

要对 首位和最后一位进行特判：

首位为0， 第二位为0， 首位种花 flower[0] = 1, cnt++

末位为0， 倒数第二位为0， 最后一位种花 flower[flower.size()-2] = 1, cnt++

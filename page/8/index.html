Veröffentlicht am 2017-08-11

hexo commands：

hexo new "postName"      # Create a new article
hexo new page "pageName" # Create a new page
hexo generate            # Generate static pages in "public"
hexo server              # Open the Preview and access the port of 4000
hexo deploy              # Deploy to github
hexo help                # View help
hexo version             # View the version of Hexo

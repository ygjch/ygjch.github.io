介绍

派生选择器允许你根据文档的上下文关系来确定某个标签的样式。通过合理地使用派生选择器，我们可以使 HTML 代码变得更加整洁。

后代选择器

后代选择器有一个容易被忽视的方面，就是两个元素的层次间隔可以是无限的。
如果写作ul em，这个语法就会选择从ul元素继承所有em元素，无论em的嵌套层次多深。

子元素选择器

比较于后代选择器，子元素选择器是希望选择某个元素的子元素，而不是选择任意后代元素。

结合后代选择器和子选择器

table.company td > p这个选择器会选择 td 元素下的 p元素，这个td元素本身从table元素继承，table元素有一个包含company的class属性。

相邻兄弟选择器

需要选择另一个元素后的元素，且两个元素由共同的父元素，可以使用相邻兄弟选择器。

如，要增加紧接在h1元素后出现的段落的上边距，可以这样写：
h1 + p {margin-top: 50px;}

相邻兄弟选择器使用加号（+），即相邻兄弟结合符。
